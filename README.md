# 📊 Customer Behavior Analysis using SQL & Python

## Overview
This project analyzes customer behavior using SQL and Python to uncover insights about spending patterns, engagement, and key business drivers.

---

## Objectives
- Identify high-value customers  
- Analyze income vs spending behavior  
- Evaluate impact of discounts  
- Understand customer engagement patterns  

---

## Tech Stack
- Python (Pandas, Matplotlib)  
- SQL (SQLite)  
- Jupyter Notebook  

---

## Key Analysis
- Customer segmentation  
- Income vs spending  
- Purchase frequency  
- Discount impact  
- Engagement vs spending  

---

## Key Insights
- High-value customers contribute most revenue  
- Income has limited impact on spending  
- Engagement (browsing time) influences spending more than discounts  
- Discounts have minimal effect on increasing revenue
